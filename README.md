# Pictora Login System

> A stylish anime-inspired login interface designed for modern web applications.

## 📸 Preview
![Pictora Login Preview](https://github.com/itsBogs/coolTo-Ne-Login/blob/816ddc27df3ee3aa5caa77101b853d7efd8f5c32/main.jpg)
---

## ✨ Features

- Responsive login interface
- Modern and minimal design
- Animated image carousel/slider
- Dark mode toggle
- Social login buttons (Google & Facebook UI only)
- Forgot password link
- Mobile-friendly layout

## 🛠️ Built With

- HTML5
- CSS3

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/Pictora-Login-System.git
```

### Run the project

Open `login.html` in your browser.

## ⚠️ Note

This project currently includes only the frontend interface. Backend functionalities such as authentication, database integration, and OAuth are not yet implemented.
